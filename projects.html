Projects

KtBukkitGui

Kotlin bukkit GUI is a library focused on providing an easy to use DSL styled wrapper for designing GUIs in Minecraft.

It also provides easy methods to create animated and static scoreboards.

PvPHub

PvPHub is a 1.17+ Minecraft PvP server aimed at providing a playerbase with the best experience that Vanilla pvp can offer.

It has a variety of servers and an active community and staff team.

Most Spigot/Bukkit projects I make are targetted to make developing for PvPHub easier.

VelocityUtils

A library for making Velocity development easier and cleaner.
Comes with it's own plugin system and plenty of DSL to make your code nicer.

Also allows for using packets on the proxy.

ProtocolLibDsl

A small and lightweight library for making ProtocolLib easier and cleaner.

The library aims at reducing the amount of code you have to write in order to achieve the same goal.

ServerGui

The ServerGui velocity plugin allows owners to easily create a gui for users to navigate their servers.

The config is easy, and guis are cached in memory in order to display them quicker for the player.

Announcer Velocity

Announcer Velocity is a plugin for velocity servers to help make scheduled announcements easy.

It comes with an intuitive GUI and the option to edit directly from the config.

Announcements are extremely configurable with delays, content, sounds and more.
